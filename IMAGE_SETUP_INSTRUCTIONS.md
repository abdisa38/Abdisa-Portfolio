# 📸 Image Setup Instructions

## Follow these steps to add your images:

### 1. Create Folders
Create this folder structure in your project:
```
Abdisa-Portfolio/
├── public/
│   └── images/
│       ├── profile.jpg
│       ├── kuraz-logo.png
│       ├── ctc-logo.png
│       ├── cert-kuraz.jpg
│       ├── cert-startup.jpg
│       └── cert-entrepreneurship.jpg
```

### 2. Save Your Images

**Profile Photo:**
- Save your red shirt profile photo as `profile.jpg`
- Location: `public/images/profile.jpg`

**Company Logos:**
- Save Kuraz Technologies logo (gold circle) as `kuraz-logo.png`
- Location: `public/images/kuraz-logo.png`
- Save CTC club logo (blue CTC) as `ctc-logo.png`
- Location: `public/images/ctc-logo.png`

**Certificates:**
- Save Kuraz internship certificate as `cert-kuraz.jpg`
- Location: `public/images/cert-kuraz.jpg`
- Save Future Startup Founders certificate as `cert-startup.jpg`
- Location: `public/images/cert-startup.jpg`
- Save Entrepreneurship training certificate as `cert-entrepreneurship.jpg`
- Location: `public/images/cert-entrepreneurship.jpg`

### 3. What Has Been Updated

✅ **About Section** - Now uses your profile photo
✅ **Experience Section** - Shows Kuraz Technologies logo
✅ **Leadership Section** - Shows CTC logo
✅ **Certifications Section** - Displays all 3 certificates with images

### 4. After Adding Images

Run the build to verify everything works:
```bash
npm run build
```

Then test locally:
```bash
npm run dev
```

Your portfolio will now show all your real images! 🎉
